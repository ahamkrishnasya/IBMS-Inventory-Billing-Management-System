# IBMS-Inventory-Billing-System
IBMS stand for Inventory & Billing Management System this application is able to manage the Inventory such as Products, Purchases, Sales, Suppliers and it's Reports which justifies the flow of Business in v1 I covered only these things but on further versions i would like to add more modules such as Stock Ledgers, which will make the Business Management more realiable.
#Software Requirement
Visual Studio 2022 
SQL Server Management Studio 2022(SSMS)
